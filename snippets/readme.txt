Snippets here are in the [VSCode Snippet Format](https://code.visualstudio.com/docs/editor/userdefinedsnippets#_snippet-syntax) unless otherwise stated.

Each package of snippets should have a VSCode extension manifest named "package.json" with a `contributes` key with a `snippets` subkey in the format defined [here](https://code.visualstudio.com/api/references/contribution-points#contributes.snippets)

The language identifiers are found [here](https://code.visualstudio.com/docs/languages/identifiers).
