# Dark Reader
[Dark Reader](https://github.com/darkreader/darkreader) is a browser extension that attempts to enable dark mode on websites that do not natively supports it.

The config here incorporates the Catppuccin Mocha colours, the colour for `::selection` is obtained by mixing Catppuccin's Blue at 30% opacity with Catppuccin's Base (this is also the percentage used by the UserStyle browser extension configs).
